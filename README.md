## Description

- A UI react application of **product list** built using **TypeScript**, enabling users to find the prodcuts in the card format. 
---

## Features

- Implemented a ProductList component that renders a list of products.
- Created a ProductCard component for individual product display.
- **Product Display**: Renders a list of products using a reusable `ProductCard` component.
- **Conditional Rendering**: Displays "Available" (green) or "Out of Stock" (red) labels based on product availability.
- **Type-Safe Code**: Built with TypeScript for enhanced reliability and easier maintenance.

## Installation

### 1. Clone the Repository

```bash
git clone https://github.com/Umamaheswari-gude/product-list-application.git
cd product-list-application
```

### 2. Install dependencies
```bash
npm install
```

### 3. Run the application 
```bash
npm start
```
- The app will be available at `http://localhost:3003`
